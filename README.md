# Online Retail Sales Analysis

## Business Question
What drives revenue for a UK based online retailer? 
Which products and countries generate the most money, 
and when do sales peak?

## Dataset
Online Retail II dataset from the UCI Machine Learning Repository.
1,067,371 transactions across two years.
Source: https://archive.ics.uci.edu/dataset/502/online+retail+ii

## Tools Used
- SQL (SQLite via DBeaver) for data cleaning and analysis
- Tableau Public for dashboard and visualisation

## Key Findings
- Top revenue country: United Kingdom
- Peak revenue month: November
- Top product overall: Regency Cakestand 3 Tier
- Top product in November: Paper Chain Kit, suggesting festive 
  decoration products drive the end of year sales peak
